# Python Escape Characters

| Code | Name            |
| ---- | --------------- |
| \n   | New Line        |
| \t   | Tab             |
| \'   | Single Quote    |
| \\   | Backslash       |
| \r   | Carriage Return |
| \b   | Backspace       |
| \f   | Form Feed       |
| \ooo | Octal value     |
| \xhh | Hex value       |

## 1. (\n) New Line
Example : 
```
>>> txt = "Abhi\nShek"
>>> print(txt)
Abhi
Shek
>>>
```
