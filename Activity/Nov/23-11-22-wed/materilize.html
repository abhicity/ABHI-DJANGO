Data Types

Number

  1. It is an integer data or floating point number

//Number Example
5, 10.5, 15e-2 etc

String

  1. String is used to store text.
  2. It represents the textuak data.
  3. It uses ' or " to store the text data.

Eg :
Single quotes : ('') 'Hello'
Double quotes : ("") Hello"
Backticks : (``) Hello

Null

  1. It denotes the null value.
  2. null is a special value that represents empty or unknown value.
  3. null is not the same as NULL or Null.
  4. The null number variable will be empty

//Null Example
var number = null;

Boolean

  1. This data type represents logical entities.
  2. In boolean there are two values true or false in lower case.
  3. It is easier to think of it as a yes/no switch.

//Boolean Example
var data_checked = true;
var value_counted = false;
console.log(value_counted) //false

Undefined

  1. It has only one value undefined.
  2. When a variable is declared but not initialized, it is assigned the value of undefined.

//Undefined Example
var name;
console.log(name); // undefined

Array

  1. Array is a collection of data.
  2. The array is a single variable that is used to store different elements.
  3. Data is defined inside [ ]
  4. Data's are seperated with ,

//Array Example
var fruits=["Apple","Mango","Orange"]
console.log(fruits)
(3) ['Apple', 'Mango', 'Orange']

0: "Apple"
1: "Mango"
2: "Orange"
length: 3

Console.log()

  1. It outputs a message to the web console.
  2. It is used to print any kind of variables.
  3. It uses ' or " to store the text data.

//Console.log() Example
var a = 2;
console.log(a); //2
